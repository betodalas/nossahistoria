═══════════════════════════════════════════════
PASSO 1 — TESTAR O APP NO NAVEGADOR
═══════════════════════════════════════════════

1. Abra o Terminal (Command + Espaço → "Terminal")

2. Navegue até a pasta frontend:
   cd ~/Downloads/nossa-historia/frontend

3. Instale as dependências (só na primeira vez):
   npm install

4. Crie o arquivo de configuração:
   cp .env.example .env

5. Rode o app:
   npm run dev

6. Abra no navegador:
   http://localhost:5173

═══════════════════════════════════════════════
PASSO 2 — PUBLICAR O FRONTEND NO VERCEL
═══════════════════════════════════════════════

1. Instale o Vercel (só uma vez):
   npm install -g vercel

2. Dentro da pasta frontend, rode:
   vercel

3. Responda as perguntas:
   - Set up and deploy? → Y
   - Which scope? → sua conta
   - Link to existing project? → N
   - Project name? → nossa-historia
   - In which directory is your code? → ./
   - Override settings? → N

4. Pronto! Vai aparecer um link tipo:
   https://nossa-historia.vercel.app ✅

═══════════════════════════════════════════════
PASSO 3 — PUBLICAR O BACKEND NO RENDER
═══════════════════════════════════════════════

1. Acesse render.com e crie uma conta grátis

2. Clique em "New +" → "Web Service"

3. Escolha "Deploy manually" (sem GitHub)

4. Preencha:
   - Name: nossa-historia-api
   - Runtime: Node
   - Build Command: npm install && npm run build
   - Start Command: npm start

5. Em "Environment Variables", adicione:
   DATABASE_URL     → (pega do banco PostgreSQL do Render)
   JWT_SECRET       → qualquer texto longo e aleatório
   PAYPAL_CLIENT_ID → do painel PayPal
   PAYPAL_SECRET    → do painel PayPal
   CLOUDINARY_URL   → do painel Cloudinary

═══════════════════════════════════════════════
PASSO 4 — GERAR O APK PARA ANDROID
═══════════════════════════════════════════════

(Faça isso depois que o backend estiver no ar)

1. Dentro da pasta frontend, instale o Capacitor:
   npm install @capacitor/core @capacitor/cli @capacitor/android

2. Inicie o Capacitor:
   npx cap init "Nossa História" "com.nossahistoria.app"

3. Gere a build do app:
   npm run build

4. Adicione o Android:
   npx cap add android

5. Sincronize:
   npx cap sync android

6. Abra no Android Studio:
   npx cap open android

7. No Android Studio:
   - Menu Build → Generate Signed Bundle/APK
   - Escolha "Android App Bundle"
   - Crie uma chave (siga o assistente)
   - Clique em Finish
   - O arquivo .aab estará em: android/app/release/

═══════════════════════════════════════════════
PASSO 5 — PUBLICAR NA PLAY STORE
═══════════════════════════════════════════════

1. Acesse play.google.com/console
2. Pague a taxa de R$ 125
3. Crie um novo app
4. Suba o arquivo .aab gerado no Passo 4
5. Preencha as informações do app
6. Adicione 20 testadores (teste interno)
7. Aguarde 14 dias de teste
8. Solicite acesso à produção
9. App no ar! 🎉
